## 📖 功能导航

### 🔥 最常用功能（推荐）

#### 1. 功能开发 - `/feature-dev`
**🎯 什么时候用**: 需要开发新功能时
**✨ 能做什么**:
- 智能分析现有代码库
- 设计功能架构方案
- 自动生成实现蓝图
- 全流程质量审查

**💡 使用示例**:
```
我想为用户管理系统添加一个新的角色权限功能
```

#### 2. 代码审查 - `/code-review`
**🎯 什么时候用**: 需要审查PR或代码质量时
**✨ 能做什么**:
- 自动化代码审查
- 多维度质量检测
- 高置信度问题筛选
- 详细改进建议

**💡 使用示例**:
```
审查我最近的代码变更
```

#### 3. Git管理 - `/commit` 或 `/commit-push-pr`
**🎯 什么时候用**: 日常Git操作时
**✨ 能做什么**:
- 智能生成提交信息
- 一键推送并创建PR
- 清理无效分支

**💡 使用示例**:
```
提交我的代码变更
```

### 📊 智能分析功能

#### 4. 项目健康分析
**🎯 什么时候用**: 需要了解项目整体状况时
**✨ 能做什么**:
- 代码复杂度分析
- 测试覆盖率检查
- 变更频率统计
- 技术债务识别

**💡 使用示例**:
```
分析这个项目的健康状况
```

#### 5. 市场分析 - `/mcp deepseek`
**🎯 什么时候用**: 需要投资或市场分析时
**✨ 能做什么**:
- 股票技术分析
- 市场趋势预测
- 投资建议提供

**💡 使用示例**:
```
/mcp deepseek analyze 000042
/mcp deepseek market "新能源汽车板块"
```
